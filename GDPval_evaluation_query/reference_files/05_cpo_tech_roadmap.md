# 05 · CPO / 光互连技术路线与节奏 — 事实性背景简报 (context brief)

> 数据性质：**中立事实背景**。本文件只陈述技术定义、路线选项与公开的厂商里程碑,**不含**任何投资结论、标的排序、配置建议或买卖观点(那是交付物 gold 的工作)。用于给分析师提供领域事实底座。里程碑日期为公开发布信息,细节以厂商官方为准。

## 1. CPO 是什么,为什么出现

共封装光学(Co-Packaged Optics, CPO / 共封装光学)= 把硅光(silicon photonics)光引擎与交换 ASIC 或 XPU **共同封装在同一基板上**,用近封装的光 I/O 取代交换机前面板上的**可插拔光模块(pluggable transceiver)**。

驱动因素(工程层面,非观点):
- **功耗墙**:AI 数据中心互连能耗以 pJ/bit(每比特皮焦耳)计;前面板可插拔在 1.6T/3.2T 世代的 SerDes 电通道损耗与重定时(retimer)功耗快速上升。CPO 缩短电通道、把光引擎移到 ASIC 旁,目标是显著降低每比特能耗与延迟。
- **带宽密度**:交换 ASIC 吞吐从 51.2T 向 102.4T 演进,前面板可容纳的可插拔端口数量受物理空间限制;CPO 用共封装提高单位面积带宽。
- **规模化(scale-up)**:GPU/XPU 纵向扩展(如 NVLink 域)对低延迟、高密度光互连的需求上升。

CPO 的取舍(事实):共封装后光引擎**不可现场热插拔**,可维护性、可靠性、良率、光源(激光器)可维修性成为工程难点;这也是**外置光源(ELS)**方案受关注的原因。

## 2. 三条并行路线(定义,非排序)

| 方案 | 定义 | 功耗/成本定位 | 可维护性 |
|------|------|----------------|----------|
| 可插拔 (pluggable, 如 OSFP/QSFP-DD, DR/FR/2xFR4) | 前面板可热插拔光模块,含完整 DSP | 成熟、可维护性最好、功耗最高 | 可现场热插拔 |
| LPO / LRO (linear-drive / linear-receive optics) | 去掉或简化模块内 DSP,由交换 ASIC 侧 SerDes 直驱,降低功耗与时延 | 介于可插拔与 CPO 之间的**低功耗过渡**,仍保留可插拔形态 | 可插拔形态 |
| CPO (co-packaged optics) | 光引擎与 ASIC 共封装,前面板无可插拔光模块 | 目标功耗最低、密度最高 | 不可热插拔(工程难点) |

外置光源(External Laser Source, ELS)/ 连续波(CW)DFB 激光器:在 CPO 与部分硅光方案中,激光器从模块内移到封装外/前面板可维修位置,单独供光。ELS 使 **CW-DFB 激光芯片**成为独立环节。

## 3. 速率节奏(公开路线,时间为行业普遍预期)

- **800G**:2023–2024 数据中心放量的主力速率(DR8/2xFR4 等)。
- **1.6T**:2025–2026 进入量产爬坡,匹配新一代 51.2T/102.4T 交换与新一代 GPU 平台。
- **3.2T**:2027+ 讨论中的下一台阶,被视为 CPO 更可能规模落地的速率世代。
- 行业普遍认为:**近端(2025–2026)AI 互连以可插拔为主力**,CPO 的规模出货多被指向 **2026–2028+**;LPO/LRO 作为过渡在此期间并行。CPO 的落地节奏(时点)在业内仍有分歧。

## 4. 厂商公开里程碑(事实,不构成观点)

- **NVIDIA**:公开发布 **Quantum-X Photonics**(InfiniBand)与 **Spectrum-X Photonics**(以太网)硅光/共封装交换平台,面向后续世代的 AI 工厂网络;GB200/GB300 等平台的 NVLink 纵向扩展提升 scale-up 互连需求。
- **Broadcom**:交换 ASIC **Tomahawk 5(51.2T)**、**Tomahawk 6(102.4T 级)**;发布 **Bailly** 51.2T CPO 交换(集成硅光引擎)作为 CPO 早期产品化里程碑;并提供可插拔/LPO 所需的光 DSP。
- **代工/封装**:**台积电 COUPE**(Compact Universal Photonic Engine)等先进封装/硅光平台被列为 CPO 光引擎的关键制造环节;先进封装(CoWoS 类)产能是共封装的约束之一。

## 5. 价值链地图(环节划分,中立)

1. **交换 ASIC / XPU**:Broadcom(Tomahawk)、NVIDIA、Marvell 定制 ASIC 等。
2. **硅光光引擎 / 光 DSP**:硅光 PIC、光引擎、相干/PAM4 DSP。
3. **光芯片 · 光源 ELS**:EML/DML、CW-DFB 激光芯片、探测器。
4. **光模块**:800G/1.6T 可插拔模块整机组装与测试。
5. **连接器 · MPO · 光纤 shuffle**:光纤阵列、MPO/MTP 连接器、fiber shuffle、光引擎用光学无源器件。
6. **先进封装 · 代工**:硅光晶圆、共封装/先进封装、测试。

> 说明:以上仅为环节与厂商的**客观归类**,不代表任何标的的推荐、权重或优劣判断。
