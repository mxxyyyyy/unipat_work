# 快速核验银行稳健性交付物的关键计算与口径，识别敷衍作答

## 核验要点（如何快速核验关键计算与口径）
**先验口径，再验数值**——银行指标最常见的错误是分母或口径错误，核对分母比核对小数点更为高效：
- **coverage = 拨备 / 不良**（而非 / 总贷款）；cost of risk = 减值 / 贷款（bps，注意是否年化）；NCO rate = 净核销 / **平均**贷款；美式 reserve coverage = ALLL/ACL / **noncurrent**（非总贷款）。
- **CET1 ratio = CET1 / RWA**（而非 / 总资产，后者是 RWA density 或杠杆概念）；leverage ratio = Tier1 / 敞口计量。
- **NIM = 净利息收入 / 平均生息资产**（分母是生息资产，非总资产）；cost/income = 费用 / 营业收入；ROE 与 ROA 用**平均**权益或资产。
- LCR = HQLA / 30 天净流出；NSFR = ASF / RSF；两者均须 ≥100% 方为达标——注意不应将「LCR = 95%」称为达标。
- **压力后指标**：depletion = 起点 CET1 − 不利情景**谷值**（bp）；buffer to MDA = 实际 CET1 − 总要求（P1 + P2R + combined buffer）。核对是否用了谷值而非期末值、是否将当期 CET1 误当作压力后。
- **对标**：EBA 口径采用**加权平均**；抽取 1–2 国或机构复算加权均值与分位位置，检查是否以简单平均或算反分位以规避核验。
- 抽样复算：从参考材料取 2–3 行，手算比率与序列变动（bps 或百分点）并与交付物对照；核对单位（%、bps、百万本币）是否一致。

## 常见问题与识别
- **口径偷换**：coverage 用总贷款做分母、CET1 用总资产、NIM 用总资产、depletion 用期末值——数值会系统性偏离，复算即可现形，判 accuracy 与相关客观项 0。
- **以简单平均冒充加权平均或分位算反**：对标基准与目标位置随之出错，判对标客观项 0。
- **静态看待不良**：只报当期 NPL，不以 Stage 2、cost of risk、迁徙或压测作前瞻，却下「资产质量稳健或恶化」的结论——前瞻判断项不给分。
- **编造参考材料之外的数据**：外部行业均值、逐笔或分组合敞口（参考材料仅有汇总时）、目标价、官方 CAMELS/SREP 评分——凡交付物给出而参考材料无据的具体数字，一律按编造扣减 accuracy 守门项。
- **承诺却未交付**：正文称「见 Summary」「见附表」「见图 3」而文件缺该工作表、章节或图，或所点名文件类型、命名或篇幅不符——对应 required 或格式项判 0。
- **结论与数据相矛盾**：综合判断方向与所给指标相反，或四支柱结论相互矛盾——structure_style_relevance 与判断项扣分。
- **空泛无据**：满纸「审慎监测」「总体可控」却无指标支撑、无对标、无关注项——判断项与相关性项低分。

## 判分提醒
- objective 与口径项严格二值：复算正确给满分，错误或缺失给 0。judgment 项**接受合理变体**——方向正确、有参考材料支撑即给分，即使措辞与 answer_key 不同；空泛、无据或方向错误方才扣分。
- 三大失败模式（不交付、忽略或编造参考数据、格式错误）对应的 required 项一旦不满足，判 0 并在 justification 中指明。
