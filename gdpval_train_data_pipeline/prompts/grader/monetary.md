# 货币政策与宏观展望：交付物评分的领域指引

## 核验要点（如何快速验证关键计算与口径）
- **口径核**：core PCE 必须除食品与能源；supercore 必须除住房；HICP core 除能源与食品；PCE 不等于 CPI。口径用错 → 该客观项判 0。
- **run-rate 复算**：3 个月 / 6 个月折年是否采用 (level_t / level_{t−k})^(12/k) − 1（或 MoM 折年复利），数值应对到 ±0.1pp；只报 YoY 却声称给出了 run-rate → 判 0。
- **利差核**：2s10s = 10y − 2y，3m10y 同理，单位为 bp，近 N 月变动的符号正确。
- **隐含路径核**：由 OIS 或期货反推的到期政策利率与隐含降息/加息 bp 是否与所给序列自洽；terminal 及时点是否有据。
- **点阵图核**：引用 SEP 是否区分 median / central tendency / range；把 median 当作区间 → 扣分。
- **实际利率核**：名义政策利率 − 通胀预期，口径一致。
- **格式核**：文件类型与命名、文字类产物约定各节、xlsx 的 `Data` / `Calc` / `Scenarios` 工作表及其列与单位、行数或情景数 N，均须与格式要求严格对齐；不符 → 相应项判 0。

## 常见敷衍作答手法与识别
- **口径张冠李戴**：CPI 冒充 PCE、headline 冒充 core、supercore 未除住房——核对定义与数值即可发现，扣 accuracy 及相关客观项。
- **只报存量、不算动能**：只给 YoY 而不算 run-rate，或数字与参考材料对不上 → accuracy 项判 0。
- **路径混同或无据**：把市场隐含路径当作 SEP，或凭空给出 terminal 而无 OIS 或点阵图支撑 → 扣判断项与 accuracy。
- **空泛方向语**：「通胀将回落、政策料转向、曲线走陡」而无分项、利差或路径证据支撑 → judgment 不给分（接受合理变体，但必须有据）。
- **编造数据**：引用参考材料中不存在的序列或读数 → accuracy 客观项扣尽，并触发「忽略参考数据」的必过项。
- **承诺未交付**：正文声称「附情景工作簿或图表」但文件中并无对应工作表或图 → 相应 objective 与 required 项判 0。
- **格式跑偏**：格式要求 xlsx 却交付 md，或工作表与列名不符 → 格式相关项判 0。
- **判断项软锚点**：只要方向正确、有参考材料支撑，即便措辞与 answer_key 不同也应给分；空泛、无据或方向错误才扣分。
