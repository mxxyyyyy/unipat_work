# 外部部门评估交付物的复核评分

## 核验要点（如何快速验证关键计算与口径）
- **BPM6 恒等式**：CA + KA 应约等于 FA（净额），残差即 NEO；符号方面，金融账户 = 净获取资产 − 净发生负债，储备增加计入资产侧为正。符号或加总错误则扣减 accuracy。
- **融资指标**：核验基本余额 = CA + 净 FDI；外部融资需求 = CA 逆差 + 剩余期限短债 + 中长期摊还。检查是否漏项或误用原始期限短债。
- **储备充足性逐口径复算**：进口覆盖 = 储备 ÷ 月均进口（对照 3 月）；Greenspan-Guidotti = 储备 ÷ **剩余期限**短债（对照 100%）；储备/M2；**ARA** = 30% × 短债 + 其他组合负债、M2、出口按汇率制度权重（浮动 15/5/5%、固定 20/10/10%），储备/ARA 对照 100–150%。核对权重是否与汇率制度匹配、短债是否采用剩余期限、达标结论是否与算出的数值一致。
- **BIS 口径**：跨境头寸"变动"是否采用汇率调整后数据（而非存量差）；CBS 外国债权（UR）= 跨境债权 + 当地债权；IC 与 UR 是否被错误相加；LBS（居民）与 CBS（国籍）是否被混用。
- **占比分母**：短期外债占比、外债分部门或分币种占比的分母是否一致（毛外债与储备）。
- **格式**：文件名、sheet、篇幅是否符合 format_spec；xlsx 测算列是否可回溯到输入行。

## 常见问题与识别方法
- **口径不实**：以存量差充当 BIS 流量、ARA 使用错误权重或忽略汇率制度、短债采用原始期限、将 IC 与 UR 相加——复算即可显现，命中则 accuracy 与相关客观项清零。
- **以单一口径下结论**：储备或脆弱性判断仅给出一个指标即下结论——判断项给低分，缺失的口径客观项给 0。
- **数据脱节**：结论（脆弱、充足、急停风险）未落到参考材料的具体数值，泛泛而谈——命中"忽略参考数据"守门项，则该必过项给 0。
- **声称完成却未交付**：正文称"见附表"或"见测算"但无对应 sheet、表或数值——"确实产出交付物"必过项给 0。
- **只描述不判断**：复述 BOP 或储备数据却无方向性判级与触发点——判断项不给分。
- **编造数值**：给出参考材料中不存在的储备、外债或流入数值——accuracy 清零，并在 justification 中点名指出。
