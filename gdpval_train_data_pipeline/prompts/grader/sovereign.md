# 主权信用与国别债务风险交付物的评分

## 核验要点（如何快速验证关键计算与口径）
用参考材料数字复算，逐项对照：
- **基本余额**＝总体余额 + 净利息支出？反算核对。**利息 / 收入**、**外债 / 出口**、**公共债务 / GDP** 一并复算。
- **雪球项＝(r−g)/(1+g)·d₋₁**：注意分母为 (1+g)、g 为名义增速；若采用 (r−g)·d 近似须已声明。
- **逐年闭合**：期末 d＝期初 d + 雪球项 − pb + sfa，逐年应闭合；只给终值、无 r / g / pb 分解则按未完成计。
- **债务稳定基本余额 pb\*＝(r−g)/(1+g)·d**：核对其对"实际 pb 是否达标"的判断。
- **储备**：进口月数＝储备 /（年货物进口 / 12）；Guidotti＝储备 / 短期外债 ≥ 1；ARA 100–150%。
- **GFN**＝财政赤字 + 摊还；GFN / GDP 对照约 15% 基准。
- **加总与符号**：需求侧贡献加总＝GDP 增速；经常账户各项加总与符号约定一致。
- **口径一致性**：债务口径（GG / PS / CG）全篇一致；名义与实际不混用（名义利率须配名义增长）；GDP 分母（本币现价）一致；财年口径统一。

## 常见敷衍作答表现及识别方法
- **不实的压力测试**：各情景债务路径与基线相同或几乎无变化，说明未真正重算，相关客观项给 0。
- **无据定性**："脆弱 / 可持续"缺比率支撑，judgment 项扣分。
- **编造数据**：出现参考材料中没有的具体数值（如 EMBI 480bp、评级 BB−、2026 增速 3.5%）并当作事实，accuracy 与相关客观项扣光。
- **黑箱债务路径**：只有终值、无雪球 / 基本余额 / sfa 分解，无法核验，按未完成计。
- **口径混用**：把中央政府债务标为一般政府；用实际利率配名义增长得出错误雪球项，属计算错误，扣分。
- **空壳交付物**：承诺的 sheet / 章节缺失或为空，扣减 required 与 instruction_following。
- **方向核对**：结论与 answer_key 方向明显相悖且无据，judgment 扣分；但**方向一致、表述不同的合理变体照常给分**，不应因未逐字命中 answer_key 而扣分。
