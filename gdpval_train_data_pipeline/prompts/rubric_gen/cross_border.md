# 外部部门评估的客观核验点、判断要素与守门条目

## 客观可核验点（objective 主干：指标、口径、公式、结构）
可直接对参考材料复算或对格式核验，务必充分设置：
- **BPM6 恒等式**：CA + KA = FA（净额）；金融账户 = 净获取资产 − 净发生负债；核验各分项加总，以及储备变动计入 FA 资产侧的符号。
- **融资指标公式**：基本余额 Basic Balance = CA + 净 FDI；外部融资需求 EFR ≈ CA 逆差 + 剩余期限短期外债 + 中长期到期摊还——核验取数与算式。
- **储备充足性口径**（逐条核验数值）：进口覆盖 = 总储备 ÷ 月均进口（阈值 ≥ 3 月）；Greenspan-Guidotti = 储备 ÷ 剩余期限短期外债（≥ 100%）；储备/M2；**IMF ARA 指标** = 按短债、其他组合负债、M2、出口加权（浮动权重 30/15/5/5%、固定权重 30/20/10/10%），储备/ARA 达标区间为 100–150%。核验是否采用了正确权重与汇率制度。
- **NIIP 与外债结构**：NIIP % GDP；短期外债占比、外债分部门（政府、央行、银行、其他、公司间借贷）、币种拆分——核验占比分母与加总。
- **BIS 口径**：跨境头寸"变动"采用**汇率调整后**数据；CBS 外国债权（UR）= 跨境债权 + 当地债权；IC 与 UR 不得混加——核验口径标注与来源。
- **EMPI**：汇率变动 − 权重 × 储备变动（权重按波动率倒数）——核验组成与符号。
- **结构与格式项**：交付物文件名、sheet 名与篇幅与 format_spec 一致；xlsx 的 `Data` 与 `Summary` sheet 存在；测算列公式正确且可回溯到输入行。

## 判断项（judgment）应覆盖的分析要素（以 answer_key 软性支撑，接受合理变体）
- **CA 融资质量判断**：逆差是否由稳定的 FDI 覆盖，还是依靠易逆转的组合或其他投资（"热钱"占比）。
- **金融账户构成解读**：Surge / Stop / Flight / Retrenchment 的归类是否合理，方向与毛额、净额口径是否使用正确。
- **储备充足性综合结论**：多口径孰强孰弱、最弱环节识别、总体是否充足的方向判断。
- **外债脆弱性**：短期外债与储备的匹配、货币错配或原罪、期限墙。
- **汇率压力与推动/拉动归因**：EMPI 方向、压力主要来自资本项还是经常项、推动因子还是拉动因子主导。
- **跨境敞口风险**：债权国与债务国集中度、IC 与 UR 差异（风险转移含义）、币种敞口。
- **总体判级与触发点**：脆弱性等级及具体下行触发情景是否有据。

## 常见流于形式的作答手法（据此设置 required 守门与负向条目）
- **只报数不测算**：抄录了储备与短债却不计算 Greenspan-Guidotti 或 ARA，或仅以单一口径应付——设置"逐口径完成测算且数值正确"必过项。
- **口径不实或混用**：以存量差充当 BIS 流量、将 IC 与 UR 相加、ARA 使用错误权重或忽略汇率制度、以原始期限短期外债替代剩余期限——设置 accuracy 守门。
- **忽略参考材料、自造数据**：脆弱性结论未落到参考材料的具体数值，泛泛而谈——设置"结论引用参考材料关键数值"必过项（对应"忽略参考数据"失败模式）。
- **声称完成却未交付**：正文称"见附表"或"见测算"却无对应 sheet 或表——设置"确实产出非空、可打开的交付物文件"必过项。
- **格式偏离**：要求 xlsx 却提交 md，sheet 或文件名不符——设置"格式与 format_spec 一致"必过项。
- **只描述不判断**：复述数据却不给出方向性结论、判级或触发点——判断项给低分。
