# 主权信用与国别债务风险任务评分细则的编写

## 客观可核验点清单（细则骨干，可从参考材料或格式要求直接核验）
**比率复算**（用参考材料数字反算，数值正确给分，错误或缺失给 0）：
- 基本（初级）财政余额＝总体余额 + 净利息支出；利息支出 / 收入。
- 外债 / 出口、外债 / GDP、公共债务 / GDP（口径与全篇一致）。
- 储备**进口月数＝储备 /（年货物进口 / 12）**；**Guidotti＝储备 / 短期外债 ≥ 1**；ARA 100–150%。
- 需求侧增长贡献加总＝实际 GDP 增速；经常账户＝贸易 + 服务 + 初次收入 + 二次收入（符号一致）。

**债务动态公式**：
- 雪球项＝**(r−g)/(1+g)·d₋₁**（分母为 (1+g)，名义配名义）。
- 逐年闭合：期末 d＝期初 d + 雪球项 − pb + sfa（分解加总须闭合到期末债务）。
- 债务稳定基本余额 **pb\*＝(r−g)/(1+g)·d**；并判断实际 pb 是否 ≥ pb\*。
- GFN＝财政赤字 + 到期摊还（+ 短期滚动）；GFN / GDP 对照约 15% 基准。

**压力测试设定**：增长冲击（基线 −约 1 个标准差）、利率冲击（加到边际发行利率）、一次性汇率贬值（抬高外币计价债务 / GDP）、或有负债冲击——每个情景须**真正重算债务路径**，而非仅更改标题。

**结构与格式**：产物含要求的章节 / sheet / 列；xlsx 的指定 sheet 名与列齐全；单位、口径、基期标注一致；数值与参考材料一致，不得编造。

## judgment（判断项）应覆盖的分析要素（以 answer_key 作软支撑，接受合理变体）
- 正确识别主要脆弱性驱动（债务上行由**雪球项主导**还是**基本赤字主导**；经常账户逆差由贸易项还是收入项驱动）。
- 债务可持续性方向判断合理（r−g 为正且 pb < pb\* → 债务上行且难以自稳）。
- 储备与外部充足性定性，且多口径（月数 / Guidotti / ARA）读数自洽。
- 风险情景选取与传导链合理（商品价格 / 利率 / 汇率 / 增长冲击对财政与外部的传导及量级）。
- 评级因子与信用观点方向有据，与所算比率一致；利差 / 估值判断（若适用）方向与信用基本面一致。
- 结论方向与 answer_key 一致，或为**有据的合理变体**（不要求与 answer_key 逐字一致）。

## 常见的敷衍作答表现（据此设置守门项与负向条目）
- 罗列数字而不作判断（有描述、无分析结论）。
- 空泛定性、缺比率支撑（称"外部部门较脆弱"却不算储备月数 / 短债覆盖）。
- 债务路径只给终值、不给 r / g / pb / 雪球分解，无法核验，按未完成扣分。
- 压力测试只改标题、不真正重算（各情景路径与基线雷同）。
- 编造参考材料中没有的数据（预测值、EMBI / CDS、评级符号）以充篇幅。
- 承诺的 sheet / 章节缺失或为空。
- 口径混用（把中央政府债务当作一般政府；用实际利率配名义增长计算雪球项）。

## required 守门项（针对三大失败模式各设一条必过项）
- 确实产出了要求的交付物文件（非空、可打开、文件名与类型正确）。
- 分析确实使用了参考材料的关键数据（债务、财政、外部至少各引用一处），非凭空编造。
- 交付物格式与 `format_spec` 一致（sheet / 列 / 章节 / 篇幅、口径标注齐全）。
