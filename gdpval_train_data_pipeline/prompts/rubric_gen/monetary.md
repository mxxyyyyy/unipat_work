# 货币政策与宏观展望：评分细则的领域指引

## 客观可核验点（细则主干，应尽量充分，可从参考材料或格式要求直接核验）
- **口径正确**：core PCE = PCE 除食品与能源；supercore = 核心服务除住房；HICP core = HICP 除能源与食品。口径用错的，设为失分项。
- **通胀计算**：YoY 与 3 个月 / 6 个月折年 run-rate 公式正确（折年 = (1 + MoM)^12 − 1，或 (level_t / level_{t−3})^4 − 1），数值与参考序列一致（容差 ±0.1pp）。
- **利差与曲线**：2s10s = 10y − 2y，3m10y 同理，单位为 bp；近 N 月变动 = 期末 − 期初，正负号正确。
- **隐含路径**：由 OIS 或期货反推的到期政策利率、隐含降息/加息 bp 与首次变动时点，与所给序列自洽。
- **点阵图口径**：引用 SEP 时须区分 median / central tendency / range，不得把 median 当作区间。
- **实际利率**：名义政策利率 − 通胀预期，口径与所减项一致。
- **结构与格式**：文字类产物是否包含约定各节（通胀分项、就业、金融条件、政策立场、风险平衡）；xlsx 是否包含点名的工作表（如 `Data` / `Calc` / `Scenarios`）与列（单位 % 或 bp）；文件名是否与格式要求一致；行数或情景数 N 是否正确。
- **单位与符号**：bp 与 %、YoY 与折年、区间上界与下界均不混淆。

## judgment（判断项）应覆盖的分析要素（以 answer_key 作软性参照，接受合理变体）
- **通胀动能方向**：headline 与 core 背离的解释、run-rate 相对 YoY 揭示的拐点、能源与基数效应的剥离是否合理。
- **驱动分解**：将 core 通胀归因到商品与服务、住房与 supercore，判断主导项是否正确。
- **劳动力传导**：V/U 比、辞职率、ECI/AHE 对服务通胀的压力方向判断是否站得住。
- **政策立场判断**：相对中性利率 r\* 的限制性程度，以及反应函数（泰勒规则、平衡法则）下的隐含立场，方向合理即可。
- **路径三分**：识别并解释基线路径、市场隐含路径与 SEP 之间的差异；对 terminal 及首次降息时点的判断有据。
- **曲线与期限溢价**：曲线形态变动的归因（政策预期与期限溢价孰主导）方向正确。
- **风险平衡**：上行/下行风险与触发条件是否具体、对称性判断是否合理。

每条判断项须在 `anchor_ref` 中引用 answer_key 的支撑片段；不做逐字硬匹配，只要方向正确、有参考材料支撑即给分。

## 常见敷衍作答手法（据此设置守门项与负向条目）
- **只报 YoY 不算 run-rate**，或声称计算了 run-rate 但无数值或数值有误 → accuracy 客观项判 0。
- **口径张冠李戴**：把 CPI 当 PCE、把 headline 当 core、supercore 未除住房 → 扣客观项。
- **路径混同**：把市场隐含路径当作 SEP，或凭空给出 terminal 而无 OIS 或点阵图依据 → 扣判断项与 accuracy。
- **空泛判断**：「通胀有望回落、政策料将转向」而无分项、run-rate 或利差支撑 → judgment 不给分。
- **忽略参考材料**：编造未提供的数值或引用不存在的序列 → accuracy 客观项扣尽。
- **必过项（required = true）**：① 确实产出所要求的交付物文件（非空、可打开、含约定各节或工作表）；② 分析确实使用了参考材料的关键序列（通胀分项、收益率、OIS）而非编造；③ 交付物格式（文件类型、命名、工作表、列）与格式要求一致。
