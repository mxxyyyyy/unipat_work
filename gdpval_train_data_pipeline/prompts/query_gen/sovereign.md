# 主权信用与国别债务风险分析任务的编写

## 职业、真实产物与典型数据形态
- `sovereign_risk_analyst`（评级机构 / 投行主权分析师）——**国别信用评估、评级委员会备忘**。以评级因子框架逐项定档，给出方向性信用观点。
- `country_desk_economist`（IMF / 世界银行国别桌面经济学家）——**Article IV 式国别简报、债务可持续性分析（DSA）**。沿增长—财政—外部—债务动态完成完整体检。
- `em_fixed_income_strategist`（新兴市场固定收益策略师）——**主权债投资观点、利差分析**。将基本面映射到利差、估值与相对价值。
- `country_risk_officer`（出口信用 / 国别风险官）——**国别限额建议备忘**。将外部、转移与兑付风险落实到限额与展望。

**参考材料的典型数据形态**（仅保留数据层，形如世界银行国别 Economic / Development Update 的统计附表）：国民账户（实际 GDP 增速、需求侧分项）、财政账户（收入 / 支出 / 余额占 GDP 比、利息支出）、国际收支（经常账户、FDI、储备）、公共与外部债务存量表、CPI、货币金融概览、"selected indicators" 表（通常为 3–5 年历史加当期）。官方 outlook 与预测、评级、风险定性均已剥离至 answer_key，不出现在参考材料中。

## 术语、口径与常用指标（任务须明确点名口径）
- **财政**：总体财政余额 / GDP、**基本（初级）余额 / GDP＝总体余额 + 净利息支出**、收入 / GDP、**利息支出 / 收入**（付息能力）。
- **债务**：公共债务 / GDP（须指明口径：一般政府 GG / 公共部门 PS / 中央政府 CG）、外债 / GDP、**外债 / 出口**、外币计价占比、平均到期期限（ATM）、非居民持有占比、**毛融资需求 GFN / GDP**（新兴市场基准约 15%）。
- **外部**：经常账户 / GDP、**外部融资需求＝经常账户逆差 + 短期外债 + 中长期摊还**、储备充足性（**进口月数**、**Guidotti–Greenspan：储备 / 短期外债 ≥ 1**、IMF **ARA 100–150%**）、REER、贸易条件。
- **债务动态**：Δd＝(r−g)/(1+g)·d₋₁ − pb + sfa；**雪球项 snowball＝(r−g)/(1+g)·d₋₁**；**债务稳定基本余额 pb\*＝(r−g)/(1+g)·d**。
- **市场（仅当参考材料含市场数据时）**：EMBIG 利差（bp over UST）、5Y CDS、**隐含违约概率 PD ≈ spread /(1−recovery)**（主权回收率惯用 25–40%）。
- **评级框架**：S&P 五因子（机构 / 经济 / 外部 / 财政＝弹性 + 债务 / 货币）；Moody's（经济 / 机构 / 财政实力 + 事件风险敏感度）；Fitch（SRM 定量 + QO 定性叠加）。

## 交付物惯用结构
- **国别信用评估 / 简报（文字类产物）**：摘要与方向性观点 → 增长与需求结构 → 通胀与货币 → 财政与公共债务 → 外部部门与储备 → 债务可持续性 / 情景 → 风险与结论建议。
- **DSA（数据工作簿）**：`Data`（原始序列）+ `Baseline`（逐年 d、r、g、pb、snowball、sfa、GFN）+ `StressTests`（增长、利率、汇率、基本余额或有负债冲击各一条路径）+ `Summary`（阈值对照与风险判读）。
- **利差分析（数据工作簿，需市场数据）**：按久期 / 评级同侪分组的利差、隐含 PD、相对价值（rich / cheap）判断。

## 领域示例（仅示范风格，请勿照搬内容）
1. **country_desk_economist × 数据工作簿**："You are an IMF country desk economist. 为该国年度磋商准备一份中期公共债务可持续性分析工作簿。① 用 `Data` 中的债务、财政、增长、利率序列构建 `Baseline` 逐年公共债务 / GDP 路径，并把每年变动分解为雪球项 (r−g)/(1+g)·d₋₁、基本余额与 stock-flow 调整；② 计算债务稳定基本余额并与实际基本余额比较；③ 在 `StressTests` 中做增长冲击、利率冲击、一次性汇率贬值三个标准情景并重算债务路径；④ 在 `Summary` 中对照 GFN / GDP 与债务 / GDP 基准给出可持续性判读。输出名为 `DSA.xlsx`，含 `Data` / `Baseline` / `StressTests` / `Summary` 四个 sheet，所有口径为一般政府、本币现价 GDP 分母。"
2. **sovereign_risk_analyst × 文字类产物（pdf）**："You are a sovereign risk analyst at a rating agency. 评级委员会下周复审该主权，请撰写一份 4–6 页 PDF 评级委员会备忘。① 逐项评估经济、外部、财政、货币四支柱，用附表数据复算利息 / 收入、外债 / 出口、储备进口月数等关键比率；② 构建财政与外部两个下行情景；③ 基于以上给出你自己的方向性信用观点与理由。输出 `RatingCommittee.pdf`，含执行摘要、四支柱评估、情景、结论建议。"

## 常见问题与规避要点
- **不得在任务中写出结论方向**："鉴于该国债务不可持续……请评估" 属于预设答案；应由完成者自行推导评级 / 展望 / 可持续性方向。
- **不得要求核对源报告的预测、评级或风险定性**：这些内容已在 answer_key 中，写出即构成泄漏。
- **不得索取单国数据附表中并不存在的信息**：逐券摊还时间表、债权人 / 币种分解、同侪国序列、EMBI / CDS 报价等——若任务依赖这些而参考材料未提供，则任务不可解。利差与隐含 PD 类任务仅当数据包含市场数据时才可出题。
- **口径必须点名**：债务口径（GG / PS / CG）、财政余额（总体 / 基本）、GDP 分母（本币现价 / 不变价、财年 vs 日历年）、经常账户符号约定；缺失则任务存在歧义。
