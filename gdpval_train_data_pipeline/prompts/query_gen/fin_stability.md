# 金融稳定与系统性风险：任务编写的领域要点

本领域任务以美联储金融稳定报告（Fed FSR）、欧洲央行金融稳定报告（ECB FSR）及国际清算银行（BIS）稳定章节的真实数据为基础，面向央行、监管与买方风控的从业者。

## 职业、真实产物与典型数据形态
- **金融稳定分析师（央行 / IMF）** → 系统性风险评估、FSR 式脆弱性章节。
  数据：季度时间序列（资产估值、部门债务 / GDP、CET1、可挤兑负债）、跨国面板、脆弱性热力图底表。
- **宏观审慎政策分析师** → 逆周期资本缓冲（CCyB）、系统性风险缓冲（SyRB）或借款人工具建议备忘。
  数据：信贷 / GDP 缺口序列、房价与房价收入比、家庭 DSTI 分布、CCyB 决策指标仪表盘。
- **压力测试建模师** → 情景压力测试摘要。
  数据：银行个体横截面（起始 CET1、RWA、PPNR、组合敞口）、宏观情景路径（GDP、失业、房价、利率、股指）、损失率参数。
- **风险研究（卖方 / 买方）** → 尾部风险简报。
  数据：利差与超额债券溢价（EBP）、期限溢价、CISS / 系统性压力指标、对冲基金杠杆与国债基差、货币市场基金 / 回购流量。

## 术语、口径与常用指标
- **Fed FSR 四支柱脆弱性框架**：①资产估值 ②企业与家庭借贷 ③金融部门杠杆 ④融资风险。任务宜落在其中一支柱，或跨支柱串联。
- **资产估值**：远期市盈率 / CAPE、公司债 OAS（IG / HY）、超额债券溢价 EBP、ACM 10 年期限溢价、商业地产资本化率（cap rate）/ 租金收益率、房价收入比、房价租金比。
- **借贷与杠杆**：非金融企业债务 / GDP、家庭债务 / GDP、偿债比率 DSR、净债务 / EBITDA、利息保障倍数 ICR、**信贷 / GDP 缺口**（Basel 口径：credit-to-GDP 减单边 HP 趋势，λ=400000）。
- **金融部门杠杆**：CET1 比率 = CET1 / RWA、一级杠杆率 = Tier1 / 总敞口、交易商杠杆、对冲基金总 / 净杠杆（国债基差交易）。
- **融资风险**：可挤兑（runnable）货币类负债 / GDP、LCR = HQLA / 30 天净流出 ≥ 100%、NSFR = ASF / RSF ≥ 100%、未保险存款占比、AFS / HTM 未实现损益、久期缺口。
- **系统性与尾部**：CISS 系统性压力综合指标、增长风险 GaR / GDP-at-Risk（金融条件下 GDP 增长分布 5% 分位）、金融周期、NBFI 影子银行窄口径、主权-银行链条（doom loop）。
- **政策工具**：CCyB（缓冲指引：缺口 ≤ 2 → 0；≥ 10 → 2.5% RWA；间线性）、SyRB、G-SIB / O-SII 附加、借款人工具 LTV / DTI / DSTI、压力资本缓冲 SCB（CCAR / DFAST）。

## 交付物惯用结构
- **FSR 式章节 / 系统性风险评估**：概览与总体判断 → 逐支柱脆弱性（估值 / 借贷 / 杠杆 / 融资）→ 近期突出风险 → 传导与放大机制（火线抛售、保证金螺旋、互联性）→ 政策含义。可附脆弱性热力图 / 仪表盘。
- **CCyB / SyRB 决策备忘**：缓冲指引指标读数 → 偏离指引的理由（信贷动能、房价、错配）→ 校准与相机决策 → 建议档位与释放条件。
- **压力测试摘要**：情景设定 → 起始状况 → 传导（PPNR、损失、RWA 迁移）→ 结果（CET1 谷值、资本缺口、是否通过）→ 结论。
- **尾部风险简报**：触发情景 → 关键指标读数 → 传导路径 → 概率 / 严重度定性刻度 → 对冲 / 仓位含义。

## 领域示例（仅示范风格与结构，不应照搬内容）
**示例 A（金融稳定分析师 · Markdown 章节）**
> "You are a financial stability analyst at a central bank, drafting the *Asset Valuations* section of the semiannual Financial Stability Report." 参考数据含 2015Q1–2024Q4 的远期市盈率、HY / IG 公司债 OAS、商业地产资本化率、房价收入比季度序列。请：(1) 计算各估值指标相对其历史分布的当前分位；(2) 识别估值最被拉伸（stretched）的两类资产并量化偏离；(3) 结合利率与风险偏好讨论回调的传导渠道；(4) 给出一句总体脆弱性定性判断（low / moderate / elevated / high）并说明依据。输出一份 800–1200 字 Markdown 章节，含一张估值分位汇总表。

此例子任务遵循 数据 → 分析 → 结论 的链条，分位与偏离由完成者自行计算，且不预告结论。

**示例 B（宏观审慎分析师 · xlsx 工作簿）**
> "You are a macroprudential analyst preparing the quarterly CCyB decision dashboard." 参考数据含 credit-to-GDP、名义 GDP、实际房价、家庭 DSTI 分布的季度序列。请输出名为 `CCyB_Dashboard.xlsx` 的工作簿，含 `Indicators`（各指标水平、单边 HP 趋势、缺口、Z-score）与 `Guide`（按 Basel 缓冲指引把 credit-to-GDP 缺口映射到 0–2.5% RWA 的建议档位）两个工作表；末列给出对偏离机械指引的定性说明。

此例为取数、计算与建表类子任务，明确点名工作表名称与列。

## 本领域常见的失效模式与泄漏风险（须规避）
- **泄漏结论**：不得在任务中写出"估值处于 elevated""建议将 CCyB 上调至 2%""该行未通过压力测试"等方向性判断——这些正是答案。任务只提供数据与算法指引。
- **看似自足实则依赖外部数据**：不应要求完成者引用参考材料中没有的当期政策利率、最新一期实际 CCyB 档位、个别机构非公开敞口、FSB 全球 NBFI 数据。此类信息要么纳入参考材料，要么不作要求。
- **无法执行的压力测试**：若参考材料仅有汇总时间序列，却要求逐行做自下而上损失模型（需 PD / LGD 参数、组合明细），则不可解。压力测试类任务必须自带情景路径与起始横截面。
- **术语堆砌而无交付物**：仅问"什么是系统性风险""解释 CCyB"属于知识问答，而非真实交付物——任务须绑定具体数据的评估、备忘或摘要。
- **口径未点明**：涉及信贷 / GDP 缺口、DSR、CET1 时，应在任务中点清计算口径（HP 的 λ、比率的分子分母），否则完成者会产生重大歧义。
