# 货币政策与宏观展望：交付物生成的领域指引

## 专业规范与惯例（据交付物类型落地）
- **通胀口径须点名并算准**：core PCE = PCE 除食品与能源；supercore = 核心服务除住房；ECB 关注 HICP 与核心 HICP。同时给出 YoY 与 3 个月 / 6 个月折年 run-rate（run-rate 更能揭示拐点），并说明基数与能源效应。
- **政策利率与路径分三层，不得混同**：① 基线路径（你的判断）；② 市场隐含路径（由 OIS 或联邦基金期货反推）；③ SEP 点阵图（引用时区分 median / central tendency / range）。terminal rate 及首次降息或加息时点须有依据。
- **曲线与利率**：2s10s、3m10y 利差以 bp 计；区分曲线变动由「政策预期」还是「期限溢价」驱动；实际政策利率 = 名义利率 − 通胀预期；相对中性利率 r\* 判断限制性程度。
- **反应函数**：可用 Taylor 规则或 balanced-approach 规则作为参照，锚定 Fed 双重使命或 ECB 2% 对称目标。
- **就业侧**：失业率（U-3/U-6）、参与率、非农就业、JOLTS 空缺、V/U 比、辞职率、ECI/AHE，用于评估服务通胀的传导。

## 常用图表与工作簿规范
- **图表**：通胀分项堆叠或贡献度图、YoY 与 run-rate 双线图、收益率曲线快照、利差时间序列、SEP 点阵散点、情景扇形图或矩阵。坐标轴、单位（% 或 bp）、图例与数据日期须齐全。
- **xlsx**：`Data`（原始序列，标注口径、单位、来源）、`Calc`（run-rate、利差、隐含路径，公式可见）、`Summary` 或 `Scenarios`（情景矩阵：触发条件 → terminal → 曲线含义）。数字使用公式而非硬编码；单位与频率标注清晰。
- **文字类（md / docx / pdf）**：一句话政策判断置顶 → 通胀评估（分项 + run-rate）→ 就业与增长 → 金融条件 → 政策立场与前瞻（三层路径）→ 风险平衡（上行/下行 + 触发条件）。

## 专家级水准的标志
- **口径零错**：core 与 headline、PCE 与 CPI、YoY 与折年、median 与区间、bp 与 % 全不混淆。
- **计算可复核**：run-rate、利差、隐含路径均由参考序列真实算出，中间结果可回溯，数值自洽（±0.1pp）。
- **判断有据且分层**：方向判断挂靠到具体分项、利差或路径证据；基线、市场与 SEP 的差异被显式对照，terminal 与时点均有依据。
- **风险对称、可证伪**：上行与下行风险各附**可观测的触发条件**，而非空泛表述。
- **忠于参考材料**：只使用所给数据；数据缺失时明确标注假设，绝不杜撰具体读数。
