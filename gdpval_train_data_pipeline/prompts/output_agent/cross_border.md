# 外部部门评估交付物的生成

## 本领域产物的专业规范与惯例
- **外部脆弱性评估**：按 ① BOP 概览与 CA 融资 → ② 金融账户组成与质量 → ③ 外债与 IIP（分部门、期限、币种）→ ④ 储备充足性（多口径）→ ⑤ 脆弱性判级与预警旗标 → ⑥ 急停与情景风险 组织，采用 BPM6 口径与符号。
- **储备充足性评估**：逐口径列表呈现（进口覆盖月数、储备/剩余期限短债即 Greenspan-Guidotti、储备/M2、IMF ARA 指标与储备/ARA%），每个口径标注阈值与达标与否，识别最弱环节，必要时与同侪对标。
- **资本流动与汇率压力简报**：非居民组合流入按债券与股票、区域拆分；以 Forbes-Warnock 毛流量口径归类（Surge / Stop / Flight / Retrenchment）；进行推动因子（全球流动性、美债、美元、VIX）与拉动因子（本国增长、利率、制度）分解；以 EMPI 或储备-汇率联动判断压力方向。
- **跨境敞口分析（BIS）**：区分 LBS（居民口径）与 CBS（国籍口径）；CBS 以最终风险 UR 为基础做敞口，外国债权（UR）= 跨境债权 + 当地债权；给出债权国与债务国集中度、币种与工具拆分；头寸变动一律采用**汇率调整后**数据以剔除估值效应。

## 惯用指标与算式（据此计算，不得自造口径）
- 基本余额 Basic Balance = CA + 净 FDI；外部融资需求 EFR ≈ CA 逆差 + 剩余期限短期外债 + 中长期到期摊还。
- 进口覆盖 = 总储备 ÷ 月均进口（阈值 ≥ 3 月）；Greenspan-Guidotti = 储备 ÷ 剩余期限短债（≥ 100%）；储备/M2。
- **IMF ARA 指标** = 30% × 短债 + 权重 × 其他组合负债 + 权重 × M2 + 权重 × 出口（浮动 15/5/5%、固定 20/10/10%），达标区间为指标的 100–150%；先依据汇率制度选定权重。
- NIIP、短期外债占比（% 毛外债或 % 储备）、CA % GDP、NEO 占毛流量比。
- EMPI = %Δ汇率 − (σe/σr) × %Δ储备 [+ (σe/σi) × Δ利率]。
- BOP 恒等式：CA + KA = FA（净额），核对后残差即 NEO。

## 图表类型（xlsx 与 pdf 惯用）
- 金融账户组成堆叠柱（FDI、组合债、组合股、其他投资、储备），观察资金稳定性；
- 储备充足性多口径条形图并标注阈值参照线；外债分期限或分部门堆叠图；
- 非居民组合流入时序（债券对股票）；CBS 敞口按对手国或币种的条形图或占比饼图。
- xlsx 惯例：`Data` sheet 复刻输入并逐列计算（公式可回溯到输入行），`Summary` sheet 给出结论；数值标注单位（USD bn、% GDP、月）。

## 质量标杆（专家级的判定）
- **口径准确**：BPM6 符号正确、剩余期限与原始期限使用得当、ARA 权重与汇率制度匹配、BIS 采用汇率调整后变动且 IC 与 UR 不混加。
- **多口径与综合判断**：储备与脆弱性判断不依赖单一指标，给出各口径达标情况、最弱环节与总体判级，并落到参考材料的具体数值。
- **有方向、有触发点**：不止于描述数据，给出脆弱性等级与具体下行触发情景（如短债滚续压力、组合流出急停）。
- **诚实标注假设**：缺少某项投入（如无其他组合负债则无法计算完整 ARA）时明确标注，并给出可测算的替代口径，绝不杜撰数字。
