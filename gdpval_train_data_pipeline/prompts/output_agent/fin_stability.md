# 金融稳定与系统性风险：交付物的专业规范与标杆

## 产物专业规范与惯例
- **FSR 式章节 / 系统性风险评估**：先给总体判断与脆弱性刻度（low / moderate / elevated / high），再按 **Fed FSR 四支柱**（资产估值 / 企业与家庭借贷 / 金融部门杠杆 / 融资风险）逐节展开，收尾谈传导放大机制与政策含义。ECB FSR 亦可按 宏观金融环境 → 市场 → 非银 → 银行业 → 审慎政策 组织。每个论断都应挂一个可核验的指标读数。
- **CCyB / SyRB 决策备忘**：缓冲指引指标读数（信贷 / GDP 缺口、房价、家庭 DSTI）→ 与机械指引的偏离及理由 → 校准 → 建议档位与释放条件。
- **压力测试摘要**：情景假设（GDP / 失业 / 房价 / 利率 / 股指路径）→ 起始状况 → 传导（PPNR、损失、RWA 迁移）→ 结果（CET1 起始 → 谷值、资本缺口、是否通过）→ 结论。
- **尾部风险简报**：触发情景 → 关键指标读数 → 传导路径 → 严重度 / 概率定性刻度 → 对冲 / 仓位含义。

## 惯用指标与口径（算对、标对单位）
- 缺口：`gap = 100·(credit/GDP) − 单边 HP 趋势(λ=400000)`；DSR：`i·D/[1−(1+i)^(−s)]/Y`。
- 比率：CET1 / RWA、Tier1 / 总敞口、LCR = HQLA / 30 天净流出、NSFR = ASF / RSF；阈值 LCR / NSFR ≥ 100%。
- 估值：远期市盈率、CAPE、公司债 OAS、超额债券溢价 EBP、ACM 期限溢价、商业地产资本化率、房价收入比；以**历史分位 / Z-score**表达"拉伸"程度。
- 系统性：CISS 系统性压力指标、GaR / GDP-at-Risk（金融条件下 5% 分位增长）、可挤兑负债 / GDP、NBFI 窄口径。
- CCyB 缓冲指引：缺口 ≤ 2 → 0；≥ 10 → 2.5% RWA；间线性。单位严格区分 % of GDP / % of RWA / bp / 倍数。

## 图表类型（以 matplotlib 生成，嵌入 pdf / docx 或作为 xlsx 图表）
- 脆弱性**热力图 / 仪表盘**（各支柱指标的分位着色）；指标**历史分位带**（当前值 vs 历史区间）；
- CET1 **瀑布图 / 路径图**（起始 → 压力谷值 → 缺口）；估值与利差**时间序列 + 阈值线**；
- 分布图（家庭 DSTI、企业杠杆分位）；缺口序列 + 缓冲档位阶梯图。

## 专家级产物的特征
- **判断有据、量化到位**：每个定性刻度都由分位、偏离、谷值等具体数字支撑，而非空泛的"风险上升"。
- **口径干净**：分子分母、单位、名义与实际、同比与环比、缺口与水平从不混淆；假设（如 HP 的 λ、起始要求）显式标注。
- **忠于参考材料**：所有数字可从参考数据复算，缺数据处**明确标注假设**而非杜撰。
- **结构专业**：遵循 FSR、决策备忘、压测摘要的惯用骨架，先结论后论证，传导机制讲清放大渠道（火线抛售、保证金螺旋、久期错配挤兑、主权-银行链条、NBFI 去杠杆）。
- **格式打磨**：xlsx 的工作表与列与 `format_spec` 完全一致、含公式或标注计算口径；pdf / docx 排版整洁，表图有题注与单位。
