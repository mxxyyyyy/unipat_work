# 产出监管审查、信用对标与拨备分析级别的银行稳健性交付物

## 产物的专业规范与惯例
**组织框架**：稳健性评估与监管备忘按**四支柱**（资本、资产质量、盈利、流动性与融资）加敏感性组织，或按 **CAMELS**（Capital / Asset quality / Management / Earnings / Liquidity / Sensitivity，1–5 级、1 最好）或 **SREP**（1–4）组织；每一支柱给出「水平、趋势、对标、判断」。
**惯用指标与口径（计算前须先确认单位与分母）**：
- 资本：CET1 = CET1/RWA、Tier1、Total、leverage ratio = Tier1/敞口、RWA density；**buffer/headroom to MDA** = 实际 CET1 − 总要求（P1 + P2R + combined buffer）；压力后 CET1 谷值与 **depletion** = 起点 − 谷值（bp）。
- 资产质量：NPL/NPE、**coverage = 拨备/不良**、cost of risk（减值/贷款，bps）、NCO rate、Stage 1/2/3（或 noncurrent、past-due 30–89 与 90+）、reserve coverage、Texas ratio；按组合（CRE、住房按揭、消费、SME）拆分。
- 盈利：ROE、ROA、**NIM = 净利息收入/平均生息资产**、cost/income、RoRWA、PPNR；区分盈利质量（NIM 驱动与拨备回拨或一次性因素）。
- 流动性与融资：LCR、NSFR、LTD、资产质押率、存款结构（未保险与无息占比）。
- 敏感性：AFS/HTM 未实现损益对 CET1 的侵蚀、IRRBB（EVE/NII 敏感性）。
**对标口径**：EBA 报告采用**敞口或资产加权平均**（非简单平均）；给出四分位、5–95 分位区间与目标体系或机构的分位位置；解读跨国或跨层的离散原因。
**图表惯例**（若产出 pdf、docx、xlsx）：KRI 时间序列折线图、国别或机构横向条形图（叠加加权均值线与分位带）、NPL 与 coverage 或 CET1 与压力后 CET1 的散点图或子弹图、贡献分解（NII、手续费、CoR 对 ROE）。

## 质量标杆（专家级与敷衍作答之别）
**专家级**：
- 计算全部可复算、口径正确（coverage 用拨备/不良、采用加权而非简单平均、depletion 用起点减谷值），单位与本币标注清楚。
- **前瞻而非静态**：以 Stage 2 占比、cost of risk、NPL 流量与迁徙、压测 depletion 判断资产质量与资本承压方向，而非仅报当期水平。
- **对标有解读**：不仅给出分位，还解释某体系落于尾部的原因（组合集中、拨备滞后、盈利偏薄、存款结构脆弱）。
- **判断内在一致**：四支柱或 CAMELS 各项结论与综合评级方向吻合，且与参考材料数据一致；明确指出关注项与触发条件（如 CET1 逼近 MDA、coverage 下滑叠加 CoR 上升）。
- 仅使用参考材料的数据；数据缺失处**显式标注假设与局限**，不编造外部均值、逐笔敞口、目标价或官方评分。
**敷衍作答（应避免）**：只搬数字不作解读、以简单平均冒充加权、口径错误、只看静态 NPL、正文承诺「见附表」而文件缺该工作表或章节、结论与数据方向相反。

## 交付前自检
- 每个子任务是否都落到交付物中？点名的工作表、章节与篇幅是否齐全？
- 关键比率是否采用标准银行口径、可复算、单位正确？加权与分位是否算对？
- 结论是否有参考材料数据支撑、四支柱之间是否自洽？文件能否打开、格式是否符合 `format_spec`？
