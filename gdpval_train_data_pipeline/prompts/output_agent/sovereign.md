# 主权信用与国别债务风险交付物的生成

## 专业规范、章节、口径与图表
**产物结构惯例**：
- **国别信用评估 / 简报（md / docx / pdf）**：执行摘要与方向性观点 → 增长与需求结构 → 通胀与货币 → 财政与公共债务 → 外部部门与储备 → 债务可持续性 / 情景 → 风险与结论建议。
- **DSA 工作簿（xlsx）**：`Data`（原始序列）→ `Baseline`（逐年 d、r、g、pb、雪球项、sfa、GFN）→ `StressTests`（各冲击独立路径）→ `Summary`（阈值对照与风险判读）。
- **利差分析（xlsx，仅当有市场数据时）**：按久期 / 评级同侪分组的利差、隐含 PD、相对价值。

**常用指标与口径**（全篇一致，务必标注）：基本余额＝总体余额 + 净利息；利息 / 收入；公共债务 / GDP（点明 GG / PS / CG）；外债 / 出口；外部融资需求＝经常账户逆差 + 短期外债 + 中长期摊还；储备进口月数、Guidotti（储备 / 短期外债 ≥ 1）、ARA 100–150%；GFN / GDP（基准约 15%）。

**债务动态算法**（编写代码时须严格遵循）：
- 雪球项＝**(r−g)/(1+g)·d₋₁**，用 (1+g) 作分母、**名义利率配名义增长**（或实际配实际，不得混用）。
- 逐年递推：dₜ＝dₜ₋₁ + (rₜ−gₜ)/(1+gₜ)·dₜ₋₁ − pbₜ + sfaₜ。
- 债务稳定基本余额 pb\*＝(r−g)/(1+g)·d。GDP 分母口径（本币现价）全表一致。

**图表类型**：债务 / GDP 路径线图（基线 + 压力扇形）、总体与基本财政余额对比柱图、经常账户与融资来源堆积图、储备充足性多指标条图、需求侧增长贡献分解堆积柱、利差时间序列（若有数据）。

## 质量标杆（专家级产物的特征）
- 每个定性判断都有**比率或计算支撑并标注口径**，不出现无据的形容词。
- 债务路径给出**逐年分解（雪球 / 基本余额 / sfa）**，而非黑箱终值；压力测试冲击设定透明、可复算。
- 风险部分给出**传导链与量级**（如"油价 −20% → 出口收入 −X% GDP → 经常账户逆差扩至 Y%"），而非罗列风险词汇。
- 诚实标注参考材料的缺口与所用假设，**绝不杜撰具体数字**（无市场数据即不编造 EMBI 与评级）。
- 表格专业：千分位、单位行、口径 / 来源脚注、基期注释齐全；方向性结论与数据自洽。
