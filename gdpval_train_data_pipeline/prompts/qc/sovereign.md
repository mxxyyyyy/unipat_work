# 主权信用与国别债务风险任务的质量审查

## representative（贴近实务）判准与反例
**贴近真实产物**（倾向判 1）：
- 情境为真实决策场景：评级委员会复审、Article IV 磋商准备、投资委员会主权债观点、国别限额年度复核。
- 用词贴合实务：使用"基本（初级）财政余额""毛融资需求 GFN""进口月数覆盖""雪球项 / r−g""债务稳定基本余额""外部融资需求"等专业表述，而非通俗化说法。
- 产物结构与篇幅符合行业惯例（信用评估 / DSA / 利差分析 / 限额备忘），子任务沿"数据 → 分析 → 结论"链条推进。

**不像真实产物**（判 0）：
- 教科书式问答："请解释什么是经常账户""主权评级如何运作"。
- 与职业无关的工作：撰写新闻稿、营销文案，或提供个人理财建议。
- 只报单一数值而不作判断，或情境空泛（"请分析这个国家的经济"）。
- 口径通俗化，例如把中央政府债务当作"国家总债务"这类外行表述。

## solvable（可解性）常见易错情形（哪些结论实际需要外部数据）
- **市场数据缺失**：EMBI / CDS / 收益率不在参考材料中，则利差分析、隐含 PD、相对价值类任务不可解。
- **明细缺失**：仅有债务总量、缺逐券摊还表 / 债权人 / 币种分解，则要求"精确 GFN 路径 / 再融资墙"不可解（基于总量的近似估算才可解）。
- **无同侪数据**：数据包仅含单国时，评级同侪对标、跨国相对价值不可解。
- **历史序列过短**：仅给 1–2 年数据，则缺 r、g 输入的债务动态分解与趋势判断不可解。
- **依赖官方预测**：IMF / 世界银行的 outlook 与 projection 属于 answer_key、不在参考材料中，因此任何"以该预测为输入"的子任务不可解；若为"核对该预测"则构成泄漏。
- 反向提示：不应因"未给出当前评级"就判定不可解——"给出你自己的方向性判断"是自足可解的。

## 常见不合格模式清单
- **泄漏（zero_leakage=0）**：任务写出评级 / 展望方向 / 风险定性（如"该国债务已不可持续"），或参考材料中残留结论句、预测值。
- **口径歧义（unambiguous=0）**：未指明债务口径（GG / PS / CG）、财政余额是总体还是基本、GDP 分母（现价 / 不变价、本币 / USD）、财年口径。
- **交付物不明确（has_deliverable=0）**：仅说"分析一下"，未给出文件类型、文件名、sheet、章节或篇幅。
- **专业难度不足（non_trivial=0）**：单点查算，如"算一下债务 / GDP""报出储备规模"。
- **拼凑（coherent=0）**：子任务彼此不服务同一产物（在债务分析中突然插入无关的汇率制度史、通胀十年回顾）。
- **数据能力错配（solvable / on_domain=0）**：数据包不含表格数据却要求产出 DSA / 利差 xlsx 工作簿。
- **不在领域（on_domain=0）**：任务实质为货币政策立场、单家银行稳健性、逐笔资本流动等其他领域主题，而非国别宏观 / 主权信用 / 债务可持续性。
