# 货币政策与宏观展望：任务质量审查的领域指引

## representative（贴近实务）判准：是否形似央行或买方的真实产物
- **形似真实产物**：以某一政策决策或数据发布节点为情境；口径明确（core PCE、supercore、2s10s、SEP median、OIS 隐含路径）；要求分项拆解、run-rate/利差/隐含路径计算、方向判断与风险平衡；产物为简报、展望或情景工作簿，对应上述四类职业的真实交付物。
- **不像真实产物（判 0）**：教科书式问答（如「什么是泰勒规则」）；纯定义或名词解释；要求「预测下个月非农的具体数值」这类并非该职业成文产物的推测题；情境泛泛、无发布节点、无明确使用对象。

## solvable（可解性）常见陷阱（下列结论实际需要参考材料以外的信息，应判 0）
- 要求给出**未来实际公布值**（下次 CPI 或非农读数、下次会议的真实决定）——非通识可解。
- 要求引用**未提供的高频或私有数据**：日内 OIS 全曲线、逐日 SOFR、非公开纪要、终端专有指标。
- 要求计算**市场隐含路径**却未提供 OIS 或期货序列；要求计算**期限溢价**却未给模型输入或现成序列；要求 supercore 却分项中不含住房拆分。
- 要求跨国对照（Fed vs ECB），而参考材料只含单一经济体数据。
- 中性利率 r\* 若需完成者自行进行结构估计而参考材料无相应输入，则不可解；除非任务仅要求引用已给的 longer-run 口径。

## zero_leakage（无泄漏）重点核查（对照 answer_key）
- answer_key 的关键结论词——如「通胀动能放缓」「已具备降息条件」「曲线将倒挂收窄」「风险偏下行」——**是否已出现在任务或参考材料中**？出现即为泄漏，判 0。
- 参考材料应只含**原始序列与事实**（PCE 分项、失业率、收益率、OIS）；若混入源报告的**评估、前瞻或风险判断**语句（属分析或结论层面），即为泄漏，判 0。

## 本领域常见不合格模式
- **口径未点名**：只说「通胀」而不区分 headline/core、YoY/run-rate → unambiguous = 0。
- **三条路径混同**：将 SEP、市场隐含、基线路径当作同一条 → coherent / unambiguous 存疑。
- **交付物含糊**：只说「写一份分析」而无文件名、结构或篇幅，或（xlsx）无工作表与列定义 → has_deliverable = 0。
- **子任务拼凑**：通胀、汇率、财政赤字、个股各问一句，不围绕同一政策产物 → coherent = 0。
- **流于形式**：单值查询或仅复述参考材料某一行 → non_trivial = 0。
- **职业错配**：让 rates_desk_analyst 写纯通胀学术综述，或让 cb_staff_economist 做交易损益分析 → on_domain = 0。
